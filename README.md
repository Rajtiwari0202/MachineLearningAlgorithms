# Machine Learning Algorithms From Scratch

A project-based journey to learn Machine Learning from first principles by implementing algorithms from scratch in Python.

The goal of this repository is not just to use machine learning libraries, but to understand the mathematics, intuition, and implementation details behind each algorithm.

## Learning Philosophy

For every algorithm, the focus is on:

* Understanding the intuition
* Learning the mathematical foundations
* Deriving important formulas
* Implementing from scratch
* Testing on small datasets
* Comparing with industry-standard libraries
* Documenting learnings and observations
