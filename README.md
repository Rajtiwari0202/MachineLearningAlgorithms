# Linear Regression From Scratch

This project implements Linear Regression from scratch using Python and Gradient Descent without relying on machine learning libraries.

## Concepts Covered

* Linear Regression
* Prediction Function (y = mx + b)
* Error Calculation
* Mean Squared Error (MSE)
* Cost Function
* Derivatives
* Gradient Descent
* Model Training
* Prediction on Unseen Data

## Files

### notes.md

Contains theoretical notes and intuition behind Linear Regression.

### gradient_descent_demo.py

A simple implementation of Gradient Descent on a small dataset.

### linear_regression_from_scratch.py

A reusable Linear Regression class with fit() and predict() methods.

## Example

Dataset:

* 1 hour → 20 marks
* 2 hours → 40 marks
* 3 hours → 60 marks
* 4 hours → 80 marks

Learned Parameters:

* m ≈ 19.8966
* b ≈ 0.3040

Prediction:

* 5 hours → 99.7870 marks
